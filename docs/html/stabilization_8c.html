void process_stabilization ( float angle_system,
float angle_commande )

Traite la stabilisation du drone.

Calcule la correction nécessaire pour stabiliser le drone en comparant l'angle actuel du système avec l'angle de commande désiré. Dont il va tirer l'erreur. Utilise un correcteur proportionnel pour générer les signaux de commande des moteurs.

Paramètres
angle_systemAngle actuel du système en degrés
angle_commandeAngle de consigne désiré en degrés
Précondition
Les moteurs et les capteurs doivent être initialisés
Postcondition
Les moteurs sont commandés selon la correction calculée
La valeur d'asservissement est mise à jour dans g_state
